Description
This project is a basic Python keylogger created for educational and cyber security learning purposes. It demonstrates how keystroke logging works to help understand potential threats and defensive measures.

Features
Captures keyboard input
Logs keystrokes to a file
Built using core Python libraries

Purpose
To study keylogging techniques so that such attacks can be identified and prevented in real-world systems.
